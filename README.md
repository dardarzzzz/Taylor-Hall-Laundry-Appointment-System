
Taylor Hall Laundry Booking System

 Project Description

The Taylor Hall laundry system currently has no organized booking process, which leads to overcrowding, long wait times, and conflicts over machine use.

This project introduces a simple digital system that allows students to book laundry time slots, verify their student ID, and receive updates. The goal is to make laundry usage more organized, fair, and convenient.

---



* Book laundry machines in advance
* Student ID verification
* Real-time availability updates
* Email reminders for bookings

---
 Technologies Used

JavaScript 
  Used for the main application logic. It allows us to structure the program using classes and keeps the project simple without extra tools.

Supabase (PostgreSQL + Auth)
  Handles the database and user authentication. It stores booking data and manages login/signup without needing a custom backend.

EmailJS
  Sends email reminders to users about their bookings directly from the frontend.

HTML5 / CSS3
  Used to build and style the user interface, including a consistent dark theme.
